Brief description:

A lot of challenges have faced any start-up and caused a lot of problems, like high cost (for buying servers, licenses,..), time and effort wasted to setup the environment and others. Because of these problems, many solutions have been raised to solve these problems. Now we can hire servers, Source Control Servers and deploy on the Cloud, the only missing part is to "DEVELOP ON THE CLOUD" that's what we aim for. Our objective is to build a platform that provides a full development system on the Cloud, containing
  • Cloud IDE.
  • Source Control Management.
  • Deployment functionalities.
Initially Python is supported, with Django framework.
